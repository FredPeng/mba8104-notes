    二、方差与标准差（Variance & Standard Deviation）

    📐 公式总结

    概率分布形式：

    Var(X) = σ² = E(X²) − [E(X)]² = Σ xᵢ²·P(xᵢ) − μ²

    频数分布形式：

    均值 x̄ = Σfᵢxᵢ / Σfᵢ
    方差 Var(X) = Σfᵢxᵢ² / Σfᵢ − x̄²

    标准差：

    σ = √Var(X)

    🎯 标准差的意义

    • 衡量数据围绕均值的离散程度（波动性/风险）
    • 标准差越大 → 数据越分散，结果越不稳定
    • 与均值同单位，便于直接比较
    • 正态分布经验法则：μ±1σ 含 68%，μ±2σ 含 95%，μ±3σ 含 99.7%

    问题 3：基于超市数据计算方差与标准差

    使用公式： Var(X) = E(X²) − [E(X)]²

    第一步：计算 E(X²) = Σ xᵢ² · P(xᵢ)

    xᵢ xᵢ² P(xᵢ) xᵢ² · P(xᵢ)
    6360.27.200
    7.556.250.316.875
    8640.16.400
    8.267.240.213.448
    9810.216.200
    E(X²) =60.123

    第二步：计算 [E(X)]²

    [E(X)]² = 7.69² = 59.1361

    第三步：求方差

    Var(X) = 60.123 − 59.1361 = 0.9869

    第四步：求标准差

    σ = √0.9869 ≈ 0.9934

    ✅ 最终结果

    指标 数值 单位
    均值 E(X)7.69百万先令
    方差 Var(X)0.9869（百万先令）²
    标准差 σ0.9934百万先令
    💡 解读：销售额平均偏离均值约 0.99 百万先令；变异系数 CV = σ/μ ≈ 12.9%， 说明销售波动属于中等偏低水平，销售较为稳定。

    引理：方差的性质（Lemma — Properties of Variance）

    设 X 为离散型随机变量，a 和 b 为任意两个常数，则：

    性质 公式 含义
    1 Var(a) = 0 常数的方差为 0（无波动）
    2 Var(aX) = a² · Var(X) 缩放变量时，方差按系数的平方变化
    3 Var(aX + b) = a² · Var(X) 加常数 b 不影响方差（平移不改变离散程度）

    💡 直观理解

    • 性质 1：常数永远等于自己，没有波动 → 方差为 0
    • 性质 2：所有数据乘以 a，偏差也乘以 a，平方后变成 a² 倍
    • 性质 3：所有数据加 b，均值也加 b，每个数据到均值的距离不变 → 方差不变

    🔁 对比：期望与方差的性质

    操作 期望 E(·) 方差 Var(·)
    常数 aE(a) = aVar(a) = 0
    乘以 aE(aX) = a·E(X)Var(aX) = a²·Var(X)
    aX + bE(aX+b) = a·E(X) + bVar(aX+b) = a²·Var(X)
    ⚠️ 关键区别：方差对常数 b 不敏感（b 消失），而对系数 a 是平方关系。

    三、概率论与应用（Probability Theory And Application）

    📐 加法法则（Addition Rule of Probability）

    P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

    📖 符号含义

    符号含义
    P(A ∪ B)事件 A 或 B 发生的概率（A 并 B，至少一个发生）
    P(A)事件 A 发生的概率
    P(B)事件 B 发生的概率
    P(A ∩ B)事件 A 和 B 同时发生的概率（A 交 B）

    💡 为什么要减去 P(A ∩ B)？

    直接加 P(A) + P(B) 会把"A 和 B 同时发生"的部分算两次，所以必须减去一次，避免重复计算。

       ┌──────────┐
   │    A     │
   │   ┌──────┼───┐
   │   │ A∩B  │   │
   │   │      │ B │
   └───┼──────┘   │
       └──────────┘

    🎯 特殊情况

    情况 条件 公式简化
    互斥事件
    (Mutually Exclusive)    		 A ∩ B = ∅ P(A ∪ B) = P(A) + P(B)
    独立事件
    (Independent)    		 A 与 B 独立 P(A ∩ B) = P(A) · P(B)

    例题：夫妻面试题（Husband & Wife Interview）

    📋 题目

    一对夫妻去面试。妻子成功的概率为 1/6，丈夫失败的概率为 2/5。 若两人都成功的概率为 1/4，求：

    1. 至少一人成功的概率
    2. 两人都不成功的概率
    ⚠️ 注意：P(W ∩ H) = 1/4 > P(W) = 1/6，数据存在数学不一致，建议向老师核实。下方按题目给定值计算。

    🪜 Step 1: Identify Events（识别事件）

    符号事件含义
    WWife is successful妻子面试成功
    HHusband is successful丈夫面试成功
    W'Wife is NOT successful妻子面试失败
    H'Husband is NOT successful丈夫面试失败

    🪜 Step 2: Write Down the Given Probabilities（写出已知概率）

    概率符号数值
    妻子成功P(W)1/6
    丈夫失败P(H')2/5
    丈夫成功 = 1 − P(H')P(H)3/5
    两人都成功P(W ∩ H)1/4

    🪜 Step 3: Determine the Required Probabilities（求解）

    (1) 至少一人成功 — P(W ∪ H)

    P(W ∪ H) = P(W) + P(H) − P(W ∩ H)
    = 1/6 + 3/5 − 1/4
    = 10/60 + 36/60 − 15/60
    = 31/60 ≈ 0.5167
    ✅ P(at least one successful) = 31/60

    (2) 两人都不成功 — P(W' ∩ H')

    "都不成功"是"至少一人成功"的对立事件：

    P(W' ∩ H') = 1 − P(W ∪ H)
    = 1 − 31/60
    = 29/60 ≈ 0.4833
    ✅ P(none successful) = 29/60

    ✔️ 验证

    两事件互补：P(至少一人成功) + P(都不成功) = 31/60 + 29/60 = 60/60 = 1 ✓

    乘法定理（Multiplication Theorem）

    📐 独立事件的联合概率

    若 A 和 B 是相互独立事件（independent events），则 A 和 B 同时发生的联合概率（joint probability）为：

    P(A ∩ B) = P(A) × P(B)

    📖 符号含义

    符号含义
    P(A ∩ B)A 和 B 同时发生的概率（联合概率）
    P(A)事件 A 发生的概率
    P(B)事件 B 发生的概率

    🔑 独立 vs 不独立

    情况乘法公式说明
    独立事件 P(A ∩ B) = P(A) · P(B) 一个事件是否发生不影响另一个
    不独立（一般情形） P(A ∩ B) = P(A) · P(B|A) 需用条件概率 P(B|A)

    💡 直观理解

    • 独立：抛硬币两次，第一次的结果不影响第二次 → 直接相乘
    • 不独立：从一副牌中连抽两张（不放回），第一张影响第二张 → 必须用条件概率
    🔗 与夫妻面试题的联系：P(W) × P(H) = 1/6 × 3/5 = 1/10， 但题目给的 P(W ∩ H) = 1/4 ≠ 1/10，说明 W 和 H 不独立， 因此不能用本定理直接相乘。

    条件概率（Conditional Probability）

    📐 定义公式

    设 A 和 B 是两个事件。在事件 B 已发生的条件下，事件 A 发生的概率记作：

    P(A | B) = P(A ∩ B) / P(B)   (P(B) > 0)

    🕐 时间直观（理解方式）

    事件角色时间
    B条件（已知发生）过去事件 (past event)
    A待求未来事件 (future event)

    → 即"已知过去发生了 B，求未来 A 发生的概率"。

    🔗 与乘法定理的联系

    P(A ∩ B) = P(A | B) · P(B) = P(B | A) · P(A)
    ⚠️ 若 A、B 独立，则 P(A | B) = P(A)（B 不影响 A）； 下方 HR 例题是不独立的一般情形，需用实际联合数据计算。

    📊 例题：人力资源部数据

    随机抽取一名员工，求相关概率。列联表（含行列合计）如下：

    Gender Promoted Not Promoted 合计
    Male103545
    Female154055
    合计2575100

    a.1) P(Male) — 员工是男性

    P(Male) = 45/100 = 0.45

    a.2) P(Female) — 员工是女性

    P(Female) = 55/100 = 0.55

    a.3) P(Promoted | Male) — 已知是男性，求被晋升

    P(Promoted | Male) = P(Promoted ∩ Male) / P(Male)
    = (10/100) / (45/100) = 10/45 = 2/9 ≈ 0.2222

    💡 技巧：条件给定 Male，相当于只看"男性这一行"（45 人），其中被晋升 10 人 → 10/45。

    b) P(Female | Not Promoted) — 已知未被晋升，求是女性

    P(Female | Not Promoted) = P(Female ∩ Not Promoted) / P(Not Promoted)
    = (40/100) / (75/100) = 40/75 = 8/15 ≈ 0.5333

    💡 技巧：条件给定 Not Promoted，相当于只看"未晋升这一列"（75 人），其中女性 40 人 → 40/75。

    📊 结果汇总

    题目表达式分数小数
    a.1P(Male)45/1000.45
    a.2P(Female)55/1000.55
    a.3P(Promoted | Male)2/9≈ 0.2222
    bP(Female | Not Promoted)8/15≈ 0.5333

    贝叶斯定理（Bayes' Theorem / Bayes' Paradigm）

    📐 公式

    两事件形式：

    P(A | B) = [ P(B | A)·P(A) ] / [ P(B | A)·P(A) + P(B | A')·P(A') ]

    一般形式（A₁, A₂, …, Aₙ 为样本空间的一个划分，互斥且穷尽）：

    P(Aᵢ | B) = [ P(B | Aᵢ)·P(Aᵢ) ] / Σₖ [ P(B | Aₖ)·P(Aₖ) ]

    📖 核心术语

    术语符号含义
    先验概率 PriorP(A)获得新证据前的初始判断
    似然 LikelihoodP(B|A)A 成立时观察到证据 B 的概率
    后验概率 PosteriorP(A|B)结合证据 B 后更新的概率
    证据 EvidenceP(B)证据出现的总概率（全概率，归一化常数）
    💡 直观：贝叶斯定理把"已知原因求结果"的 P(B|A)，反推成"已知结果求原因"的 P(A|B)。 分母 P(B) 用全概率公式 Σ P(B|Aₖ)·P(Aₖ) 展开。

    📝 例题：谁生产了次品报告？

    财务部的 Jane、Tom、Mary 入职时经验和资质相同，接受相同培训后， 三人的次品率（LAT）分别为 2%、4%、7%。 某天从三人产出的报告中发现一份次品，求这份次品分别由谁生产的概率。

    🪜 Step 1: 识别事件

    • J / T / M = 报告由 Jane / Tom / Mary 生产
    • D = 报告是次品（defective）

    🪜 Step 2: 写出已知概率

    先验（产量均等）：P(J) = P(T) = P(M) = 1/3

    似然（次品率）：P(D|J) = 0.02，P(D|T) = 0.04，P(D|M) = 0.07

    🪜 Step 3: 求解

    先求全概率 P(D)：

    P(D) = P(D|J)P(J) + P(D|T)P(T) + P(D|M)P(M)
    = ⅓(0.02 + 0.04 + 0.07) = ⅓(0.13) ≈ 0.0433

    再用贝叶斯定理求后验（先验相等，1/3 约掉 → 正比于次品率 2:4:7）：

    谁计算后验概率百分比
    Jane0.02 / 0.13 = 2/13≈ 0.153815.38%
    Tom0.04 / 0.13 = 4/13≈ 0.307730.77%
    Mary0.07 / 0.13 = 7/13≈ 0.538553.85%
    ✅ P(M|D) > P(T|D) > P(J|D) → 次品最有可能由 Mary 生产（≈ 53.85%）， 其次 Tom（30.77%），最后 Jane（15.38%）。

    ✔️ 验证：2/13 + 4/13 + 7/13 = 13/13 = 1

    💡 关键洞察：因三人产量相同（先验相等），后验概率完全由次品率决定， 比例恰为 2 : 4 : 7。

    📊 方法三：表格法（Tabular Method）

    解贝叶斯题最系统、最不易出错的方法——列一张 5 列表，逐列计算：

    事件 Aᵢ 先验 P(Aᵢ) 条件 P(B|Aᵢ) 联合 P(Aᵢ∩B) 后验 P(Aᵢ|B)
    A₁ (Jane)1/30.020.02/3(0.02/3)/(0.13/3) = 2/13
    A₂ (Tom)1/30.040.04/3(0.04/3)/(0.13/3) = 4/13
    A₃ (Mary)1/30.070.07/3(0.07/3)/(0.13/3) = 7/13
    合计1—P(B) = 0.13/31

    🪜 各列怎么算

    列来源说明
    ① 先验 P(Aᵢ)题目给定各来源被选中的概率
    ② 条件 P(B|Aᵢ)题目给定各来源的次品率
    ③ 联合 P(Aᵢ∩B)① × ②乘法定理
    ④ 第③列求和Σ = P(B)全概率（= 0.13/3）
    ⑤ 后验 P(Aᵢ|B)③ ÷ P(B)贝叶斯定理
    💡 表格法优点：第③列求和直接得到全概率 P(B)（贝叶斯分母）； 后验 = 第③列 ÷ 该列合计；最后一列加起来必须 = 1（自带验证）。 结果一致：Mary 7/13 > Tom 4/13 > Jane 2/13。

    🔄 贝叶斯思维流程

    先验概率 (Prior)  →  新信息 (New Information)  →  应用贝叶斯定理  →  后验概率 (Posterior)

    已有事件的先验概率估计，获得新信息后，用贝叶斯定理计算修正后的（后验）概率。

    📝 例题：两供应商问题（Two-Supplier Problem）

    某制造厂从两家供应商进货：

    • A₁ = 零件来自供应商 1，A₂ = 零件来自供应商 2
    • 进货比例：P(A₁) = 0.65，P(A₂) = 0.35
    • G = 零件合格，B = 零件次品
    供应商合格率 %（Good）次品率 %（Bad）
    Supplier 1982
    Supplier 2955

    即条件概率：P(G|A₁)=0.98, P(B|A₁)=0.02, P(G|A₂)=0.95, P(B|A₂)=0.05

    🌳 概率树（Probability Tree）

    每个实验结果是两个事件的交集，沿树枝相乘即得联合概率：P(Aᵢ∩B) = P(Aᵢ)·P(B|Aᵢ)

    🔴 红色分支 = 次品（Bad）路径。两条红色叶子相加 = P(B) = 0.0130 + 0.0175 = 0.0305。

    📋 联合概率汇总

    路径计算联合概率
    P(A₁∩G)0.65 × 0.980.6370
    P(A₁∩B)0.65 × 0.020.0130
    P(A₂∩G)0.35 × 0.950.3325
    P(A₂∩B)0.35 × 0.050.0175
    合计—1.0000

    💡 四条叶子概率之和 = 1（树形图的自带验证）。

    ❓ 问题：一个次品弄坏了机器，它来自供应商 1 的概率是多少？

    先求次品总概率（全概率）：

    P(B) = P(A₁∩B) + P(A₂∩B) = 0.0130 + 0.0175 = 0.0305

    应用贝叶斯定理：

    P(A₁|B) = P(A₁)P(B|A₁) / [P(A₁)P(B|A₁) + P(A₂)P(B|A₂)]
    = (0.65×0.02) / [(0.65×0.02) + (0.35×0.05)]
    = 0.0130 / 0.0305 ≈ 0.4262
    P(A₂|B) = 0.0175 / 0.0305 ≈ 0.5738
    ✅ 虽然供应商 1 供货更多（65%），但因其次品率低，这个次品更可能来自 供应商 2（≈ 57.38%）。

    📊 表格法（Tabular Approach）

    (1) 事件 Aᵢ (2) 先验 P(Aᵢ) (3) 条件 P(B|Aᵢ) (4) 联合 P(Aᵢ∩B) (5) 后验 P(Aᵢ|B)
    A₁0.650.020.01300.0130/0.0305 = 0.4262
    A₂0.350.050.01750.0175/0.0305 = 0.5738
    合计1.00—P(B) = 0.03051.0000

    📐 贝叶斯定理（n 个事件的一般形式）

    P(Aᵢ|B) = [ P(Aᵢ)P(B|Aᵢ) ] / [ P(A₁)P(B|A₁) + P(A₂)P(B|A₂) + … + P(Aₙ)P(B|Aₙ) ]

    📝 练习题 41：咨询公司投标

    一家咨询公司为一个大型咨询合同投标，管理层认为中标机会是 50-50。 随后招标方要求提供补充资料。经验表明：在成功中标的投标中有 75% 被要求补充资料， 在失败的投标中有 40% 被要求补充资料。

    • (a) 中标的先验概率（在被要求补充资料之前）？
    • (b) 已知最终中标，被要求补充资料的条件概率？
    • (c) 已知被要求补充资料，求中标的后验概率？
    📌 参考解答

    设 S = 中标成功，U = 失败，R = 被要求补充资料

    (a) 先验概率 P(S) = 0.50（"50-50"）

    (b) 条件概率 P(R|S) = 0.75（题目给定）

    (c) 用贝叶斯定理（P(S)=P(U)=0.5, P(R|S)=0.75, P(R|U)=0.40）：

    P(S|R) = P(R|S)P(S) / [P(R|S)P(S) + P(R|U)P(U)]
    = (0.75×0.5) / [(0.75×0.5) + (0.40×0.5)]
    = 0.375 / (0.375 + 0.20)
    = 0.375 / 0.575 ≈ 0.6522
    ✅ 被要求补充资料后，中标概率从先验的 50% 上升到约 65.22% —— 因为"要补充资料"在成功投标中更常见，这是一个有利信号。

    案例研究：The Case of the Body in the Bag（袋中尸体案）

    正态分布（The Normal Distribution）

    📐 概率密度函数（PDF）

    f(x) = 1 / (σ√(2π)) · e^[ −(x−μ)² / (2σ²) ]

    记作 X ~ N(μ, σ²)，μ = 均值，σ = 标准差。

    🔔 关键特征

    特征说明
    形状钟形曲线，关于 μ 对称
    中心均值 = 中位数 = 众数 = μ
    范围x ∈ (−∞, +∞)，曲线下总面积 = 1
    离散度由 σ 控制，σ 越大曲线越扁平

    📊 经验法则（68–95–99.7 Rule）

    范围包含数据比例
    μ ± 1σ≈ 68%
    μ ± 2σ≈ 95%
    μ ± 3σ≈ 99.7%

    🎯 标准化：Z 分数

    把任意正态分布转化为标准正态分布 Z ~ N(0, 1)：

    Z = (x − μ) / σ

    Z 表示某个值距离均值有几个标准差，再查 Z 表得到左侧面积 P(Z ≤ z)。

    正态分布 ↔ 标准正态分布的换算

    🔄 两个方向的换算公式

    方向公式用途
    正态 → 标准正态（标准化）Z = (X − μ) / σ已知原始分，求 Z 去查表
    标准正态 → 正态（反标准化）X = μ + Zσ已知 Z（反查表得到），还原成原始分

    两公式是同一等式的移项：Z = (X−μ)/σ 两边乘 σ 再加 μ，即得 X = μ + Zσ。

    📊 X 轴与 Z 轴一一对应（μ=60, σ=12）

    同一条曲线的两套刻度：原始分 60 ↔ Z 的 0，72 ↔ 1，84 ↔ 2……每个原始分都对应唯一的 Z。

    📖 换算的本质（两步坐标变换）

    Z = (1/σ) × (X − μ)   ←  ① 平移（减 μ，中心移到 0）  ② 缩放（除 σ，单位变成"几个标准差"）

    ✏️ 用分班测试数据演示

    正向（求某分数的 Z）：X = 84 ⇒ Z = (84−60)/12 = +2（高于均值 2 个标准差）

    反向（已知 Z 求分数）：Z = 0.52 ⇒ X = 60 + 0.52×12 = 66.24

    🧭 解题时怎么选方向

    题目类型已知 → 求用哪个公式
    "分数高于/低于…的比例"分数 → 概率先 Z=(X−μ)/σ，再查表
    "前/后 X% 的分数线"概率 → 分数先反查表得 Z，再 X=μ+Zσ

    例题：Strathmore 数学分班测试（正态分布应用）

    题目：Strathmore 商学院数学分班测试，平均分 60%，标准差 12%， 假设成绩服从正态分布。

    1. 求成绩满足以下条件的考生比例：(1) 高于 50%；(2) 低于 80%；(3) 在 60%–90% 之间
    2. 求及格线，使 70% 的考生不及格
    3. 求及格线，使 65% 的考生及格

    🪜 Step 1: 用数学语言写出已知

    设 X = 考生分数（%），则 X ~ N(μ, σ²) = N(60, 12²)，μ = 60，σ = 12。

    题文字数学表达
    1-1高于 50%P(X > 50)
    1-2低于 80%P(X < 80)
    1-3在 60%–90% 之间P(60 < X < 90)
    270% 不及格的及格线 k求 k 使 P(X < k) = 0.70
    365% 及格的及格线 k求 k 使 P(X > k) = 0.65

    🪜 Step 2: 标准化随机变量

    Z = (X − μ) / σ = (X − 60) / 12
    题边界 XZ = (X−60)/12Z 值
    1-150(50−60)/12−0.83
    1-280(80−60)/121.67
    1-360(60−60)/120
    1-390(90−60)/122.5

    🪜 Step 3: 查 Z 表求解

    1-1) P(X > 50) = P(Z > −0.83)

    = 1 − P(Z ≤ −0.83) = 1 − 0.20327 = 0.7967

    → 约 79.67% 的考生得分高于 50%。

    1-2) P(X < 80) = P(Z ≤ 1.67)

    = 0.9525

    → 约 95.25% 的考生得分低于 80%。

    1-3) P(60 < X < 90) = P(0 < Z < 2.5)

    = P(Z ≤ 2.5) − P(Z ≤ 0) = 0.99379 − 0.5000 = 0.4938

    → 约 49.38% 的考生得分在 60%–90% 之间。

    💡 区分两类题：上面是"已知分数 → 求比例"（正向：标准化后查表）； 下面第 2、3 题是"已知比例 → 求分数"（反向：先反查 Z，再用 X = μ + Zσ 还原）。

    2) 及格线 k，使 70% 不及格

    70% 不及格 → P(X < k) = 0.70。反查表：面积 0.70 对应 Z ≈ 0.52（表值 0.69847）。

    k = μ + Zσ = 60 + 0.52 × 12 = 60 + 6.24 = 66.24%
    ✅ 及格线定在约 66.24%，则约 70% 的考生不及格。

    3) 及格线 k，使 65% 及格

    65% 及格 → P(X > k) = 0.65 → P(X < k) = 0.35。反查表：面积 0.35 对应 Z ≈ −0.39（表值 0.34827）。

    k = μ + Zσ = 60 + (−0.39) × 12 = 60 − 4.68 = 55.32%
    ✅ 及格线定在约 55.32%，则约 65% 的考生及格。

    四、复习题（Probability Revision Questions）

    来源：Questions.pptx — 课堂复习幻灯片

    📝 Question 1：购车问题

    John 的家庭决定购买一辆新车，候选车型为 Ford、Chevrolet 或 Toyota。

    • P(买 Ford) = 0.3
    • P(买 Chevrolet) = 0.4

    (i) 求他们购买 Ford 或 Chevrolet 的概率

    (ii) 求他们购买 Toyota 的概率

    (iii) 已知：P(Ford) = 0.3，P(station wagon) = 0.2，P(Ford station wagon) = 0.5 （⚠️ 注：此处 0.5 似为数据异常，可能应为 0.05），求 P(买 Ford 或 station wagon 或两者)

    📌 参考解法

    (i) Ford 和 Chevrolet 是互斥事件（一次只能买一辆）：

    P(F ∪ C) = P(F) + P(C) = 0.3 + 0.4 = 0.7

    (ii) 三个事件互斥且覆盖所有可能：

    P(T) = 1 − P(F) − P(C) = 1 − 0.3 − 0.4 = 0.3

    (iii) 用加法法则。⚠️ 注意 P(Ford station wagon) = 0.5 不可能 （联合概率不能大于 P(Ford)=0.3），几乎可以确定是 0.05 的笔误。按 0.05 计算：

    P(F ∪ W) = P(F) + P(W) − P(F ∩ W)
    = 0.3 + 0.2 − 0.05 = 0.45

    （若硬按题面 0.5 代入会得到 0.3+0.2−0.5 = 0，显然不合理，故判定为笔误。）

    📝 Question 2：Patricia 面试评估

    某医院考虑 Patricia 担任财务管理关键职位。面试和审核后， 人事经理认为 Patricia 表现良好的概率为 75%，表现不佳的概率为 25%。 根据其他公司的经验，咨询公司在评估财务经理时的条件概率如下：

    实际表现 咨询公司预测「会表现好」 咨询公司预测「不会表现好」
    Manager does well0.90.1
    Manager does NOT do well0.20.8

    (i) 若咨询公司预测 Patricia 表现良好，求该预测正确的概率

    (ii) 若咨询公司预测 Patricia 表现不佳，求该预测正确的概率

    📌 参考解法（贝叶斯定理）

    设 W = Patricia 实际表现好；P = 咨询公司预测 W

    P(W) = 0.75, P(W') = 0.25, P(P|W) = 0.9, P(P|W') = 0.2

    (i) P(W|P) = P(P|W)·P(W) / [P(P|W)·P(W) + P(P|W')·P(W')]
    = (0.9 × 0.75) / (0.9 × 0.75 + 0.2 × 0.25)
    = 0.675 / 0.725 ≈ 0.931

    (ii) P(W'|P') = P(P'|W')·P(W') / [P(P'|W')·P(W') + P(P'|W)·P(W)]
    = (0.8 × 0.25) / (0.8 × 0.25 + 0.1 × 0.75)
    = 0.20 / 0.275 ≈ 0.727

    📝 Case：蓝色出租车肇事案（Bayes' Analysis）

    本案例改编自 S. Salop, "Evaluating Uncertain Evidence" with Sir Thomas Bayes, Journal of Economic Perspectives, Summer 1987.

    1994 年 3 月 1 日，一名男子在过马路时被一辆超速出租车撞倒。 一名目击证人作证说她认为肇事出租车（未停车）是蓝色的。 男子起诉 Blue Cab Company 索赔医疗费用。

    该城市只有两家出租车公司：Blue Cab 和 Green Cab。

    • Green Cab 占该市出租车的 85%，Blue Cab 占 15%
    • 目击证人在类似条件下正确识别出租车颜色的可靠率为 80%

    原告律师据此认定：男子极有可能被蓝色出租车撞到。

    Required：使用贝叶斯分析，律师的结论是否正确？

    📌 参考解法

    设 B = 蓝车肇事，G = 绿车肇事，W = 证人说"蓝"

    P(B) = 0.15, P(G) = 0.85, P(W|B) = 0.8, P(W|G) = 0.2

    P(B|W) = P(W|B)·P(B) / [P(W|B)·P(B) + P(W|G)·P(G)]
    = (0.8 × 0.15) / (0.8 × 0.15 + 0.2 × 0.85)
    = 0.12 / (0.12 + 0.17)
    = 0.12 / 0.29 ≈ 0.414
    🎯 结论：实际是蓝车肇事的后验概率只有约 41.4%， 仍小于 50%！律师的结论不正确。这就是著名的"基础率谬误"—— 虽然证人 80% 可靠，但因为 Blue Cab 本身就少，整体上更可能是绿车被误认。

    📝 Exercise：疾病诊断测试

    某疾病的诊断测试有以下特性：

    • 对实际患病者：正确检测出疾病的概率为 90%
    • 对未患病者：测试报告"未患病"的概率为 0.9
    • 人群中仅 1% 患有该疾病

    从人群中随机选一人接受测试，若测试结果显示她患病， 问：她实际患病的条件概率是多少？

    ✅ 解法（贝叶斯定理）

    定义事件：

    • D = 患有该疾病
    • H = 测试结果显示患病

    已知：

    • P(H|D) = 0.9
    • P(D) = 0.01，P(D') = 0.99
    • P(H|D') = 1 − 0.9 = 0.1（假阳性率）

    应用贝叶斯公式：

    P(D|H) = [P(H|D)·P(D)] / [P(H|D)·P(D) + P(H|D')·P(D')]
    = (0.9 × 0.01) / (0.9 × 0.01 + 0.1 × 0.99)
    = 0.009 / (0.009 + 0.099)
    = 0.009 / 0.108
    = 1/12 ≈ 0.0833
    🎯 即使测试结果显示患病，实际患病概率也只有约 8.33%。 这同样是基础率谬误的经典例子——疾病罕见时，假阳性主导结果。

    五、核心知识点

    期望值 Expected Value

    • 定义：随机变量在长期试验中的平均结果，E(X) = Σ xᵢ·P(xᵢ)
    • 线性性质：E(aX) = a·E(X)，E(aX + b) = a·E(X) + b
    • 概率验证：所有概率之和必须等于 1

    方差与标准差 Variance & Std. Deviation

    • 方差 Var(X)：衡量离散程度，Var(X) = E(X²) − [E(X)]²
    • 标准差 σ：σ = √Var(X)，与原数据同单位，实际分析中常用
    • 变异系数 CV：CV = σ/μ，用于比较不同尺度数据的相对波动
    • 线性性质：Var(aX) = a²·Var(X)，Var(aX + b) = a²·Var(X)
