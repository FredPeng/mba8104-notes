    二、方差与标准差（Variance & Standard Deviation）

    📐 公式总结

    概率分布形式：

    Var(X) = σ² = E(X²) − [E(X)]² = Σ xᵢ²·P(xᵢ) − μ²

    频数分布形式：

    均值 x̄ = Σfᵢxᵢ / Σfᵢ
    方差 Var(X) = Σfᵢxᵢ² / Σfᵢ − x̄²

    标准差：

    σ = √Var(X)

    🎯 标准差的意义

    • 衡量数据围绕均值的离散程度（波动性/风险）
    • 标准差越大 → 数据越分散，结果越不稳定
    • 与均值同单位，便于直接比较
    • 正态分布经验法则：μ±1σ 含 68%，μ±2σ 含 95%，μ±3σ 含 99.7%

    问题 3：基于超市数据计算方差与标准差

    引理：方差的性质（Lemma — Properties of Variance）

    设 X 为离散型随机变量，a 和 b 为任意两个常数，则：

    性质 公式 含义
    1 Var(a) = 0 常数的方差为 0（无波动）
    2 Var(aX) = a² · Var(X) 缩放变量时，方差按系数的平方变化
    3 Var(aX + b) = a² · Var(X) 加常数 b 不影响方差（平移不改变离散程度）

    💡 直观理解

    • 性质 1：常数永远等于自己，没有波动 → 方差为 0
    • 性质 2：所有数据乘以 a，偏差也乘以 a，平方后变成 a² 倍
    • 性质 3：所有数据加 b，均值也加 b，每个数据到均值的距离不变 → 方差不变

    🔁 对比：期望与方差的性质

    操作 期望 E(·) 方差 Var(·)
    常数 aE(a) = aVar(a) = 0
    乘以 aE(aX) = a·E(X)Var(aX) = a²·Var(X)
    aX + bE(aX+b) = a·E(X) + bVar(aX+b) = a²·Var(X)
    ⚠️ 关键区别：方差对常数 b 不敏感（b 消失），而对系数 a 是平方关系。

    三、概率论与应用（Probability Theory And Application）

    📐 加法法则（Addition Rule of Probability）

    P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

    📖 符号含义

    符号含义
    P(A ∪ B)事件 A 或 B 发生的概率（A 并 B，至少一个发生）
    P(A)事件 A 发生的概率
    P(B)事件 B 发生的概率
    P(A ∩ B)事件 A 和 B 同时发生的概率（A 交 B）

    💡 为什么要减去 P(A ∩ B)？

    直接加 P(A) + P(B) 会把"A 和 B 同时发生"的部分算两次，所以必须减去一次，避免重复计算。

       ┌──────────┐
   │    A     │
   │   ┌──────┼───┐
   │   │ A∩B  │   │
   │   │      │ B │
   └───┼──────┘   │
       └──────────┘

    🎯 特殊情况

    情况 条件 公式简化
    互斥事件
    (Mutually Exclusive)    		 A ∩ B = ∅ P(A ∪ B) = P(A) + P(B)
    独立事件
    (Independent)    		 A 与 B 独立 P(A ∩ B) = P(A) · P(B)

    例题：夫妻面试题（Husband & Wife Interview）

    📋 题目

    一对夫妻去面试。妻子成功的概率为 1/6，丈夫失败的概率为 2/5。 若两人都成功的概率为 1/4，求：

    1. 至少一人成功的概率
    2. 两人都不成功的概率
    ⚠️ 注意：P(W ∩ H) = 1/4 > P(W) = 1/6，数据存在数学不一致，建议向老师核实。下方按题目给定值计算。

    🪜 Step 1: Identify Events（识别事件）

    符号事件含义
    WWife is successful妻子面试成功
    HHusband is successful丈夫面试成功
    W'Wife is NOT successful妻子面试失败
    H'Husband is NOT successful丈夫面试失败

    🪜 Step 2: Write Down the Given Probabilities（写出已知概率）

    概率符号数值
    妻子成功P(W)1/6
    丈夫失败P(H')2/5
    丈夫成功 = 1 − P(H')P(H)3/5
    两人都成功P(W ∩ H)1/4

    🪜 Step 3: Determine the Required Probabilities（求解）

    (1) 至少一人成功 — P(W ∪ H)

    P(W ∪ H) = P(W) + P(H) − P(W ∩ H)
    = 1/6 + 3/5 − 1/4
    = 10/60 + 36/60 − 15/60
    = 31/60 ≈ 0.5167
    ✅ P(at least one successful) = 31/60

    (2) 两人都不成功 — P(W' ∩ H')

    "都不成功"是"至少一人成功"的对立事件：

    P(W' ∩ H') = 1 − P(W ∪ H)
    = 1 − 31/60
    = 29/60 ≈ 0.4833
    ✅ P(none successful) = 29/60

    ✔️ 验证

    两事件互补：P(至少一人成功) + P(都不成功) = 31/60 + 29/60 = 60/60 = 1 ✓

    乘法定理（Multiplication Theorem）

    📐 独立事件的联合概率

    若 A 和 B 是相互独立事件（independent events），则 A 和 B 同时发生的联合概率（joint probability）为：

    P(A ∩ B) = P(A) × P(B)

    📖 符号含义

    符号含义
    P(A ∩ B)A 和 B 同时发生的概率（联合概率）
    P(A)事件 A 发生的概率
    P(B)事件 B 发生的概率

    🔑 独立 vs 不独立

    情况乘法公式说明
    独立事件 P(A ∩ B) = P(A) · P(B) 一个事件是否发生不影响另一个
    不独立（一般情形） P(A ∩ B) = P(A) · P(B|A) 需用条件概率 P(B|A)

    💡 直观理解

    • 独立：抛硬币两次，第一次的结果不影响第二次 → 直接相乘
    • 不独立：从一副牌中连抽两张（不放回），第一张影响第二张 → 必须用条件概率
    🔗 与夫妻面试题的联系：P(W) × P(H) = 1/6 × 3/5 = 1/10， 但题目给的 P(W ∩ H) = 1/4 ≠ 1/10，说明 W 和 H 不独立， 因此不能用本定理直接相乘。

    条件概率（Conditional Probability）

    📐 定义公式

    设 A 和 B 是两个事件。在事件 B 已发生的条件下，事件 A 发生的概率记作：

    P(A | B) = P(A ∩ B) / P(B)   (P(B) > 0)

    🕐 时间直观（理解方式）

    事件角色时间
    B条件（已知发生）过去事件 (past event)
    A待求未来事件 (future event)

    → 即"已知过去发生了 B，求未来 A 发生的概率"。

    🔗 与乘法定理的联系

    P(A ∩ B) = P(A | B) · P(B) = P(B | A) · P(A)
    ⚠️ 若 A、B 独立，则 P(A | B) = P(A)（B 不影响 A）； 下方 HR 例题是不独立的一般情形，需用实际联合数据计算。

    贝叶斯定理（Bayes' Theorem / Bayes' Paradigm）

    📐 公式

    两事件形式：

    P(A | B) = [ P(B | A)·P(A) ] / [ P(B | A)·P(A) + P(B | A')·P(A') ]

    一般形式（A₁, A₂, …, Aₙ 为样本空间的一个划分，互斥且穷尽）：

    P(Aᵢ | B) = [ P(B | Aᵢ)·P(Aᵢ) ] / Σₖ [ P(B | Aₖ)·P(Aₖ) ]

    📖 核心术语

    术语符号含义
    先验概率 PriorP(A)获得新证据前的初始判断
    似然 LikelihoodP(B|A)A 成立时观察到证据 B 的概率
    后验概率 PosteriorP(A|B)结合证据 B 后更新的概率
    证据 EvidenceP(B)证据出现的总概率（全概率，归一化常数）
    💡 直观：贝叶斯定理把"已知原因求结果"的 P(B|A)，反推成"已知结果求原因"的 P(A|B)。 分母 P(B) 用全概率公式 Σ P(B|Aₖ)·P(Aₖ) 展开。

    📝 例题：谁生产了次品报告？

    财务部的 Jane、Tom、Mary 入职时经验和资质相同，接受相同培训后， 三人的次品率（LAT）分别为 2%、4%、7%。 某天从三人产出的报告中发现一份次品，求这份次品分别由谁生产的概率。

    🪜 Step 1: 识别事件

    • J / T / M = 报告由 Jane / Tom / Mary 生产
    • D = 报告是次品（defective）

    🪜 Step 2: 写出已知概率

    先验（产量均等）：P(J) = P(T) = P(M) = 1/3

    似然（次品率）：P(D|J) = 0.02，P(D|T) = 0.04，P(D|M) = 0.07

    🪜 Step 3: 求解

    先求全概率 P(D)：

    P(D) = P(D|J)P(J) + P(D|T)P(T) + P(D|M)P(M)
    = ⅓(0.02 + 0.04 + 0.07) = ⅓(0.13) ≈ 0.0433

    再用贝叶斯定理求后验（先验相等，1/3 约掉 → 正比于次品率 2:4:7）：

    谁计算后验概率百分比
    Jane0.02 / 0.13 = 2/13≈ 0.153815.38%
    Tom0.04 / 0.13 = 4/13≈ 0.307730.77%
    Mary0.07 / 0.13 = 7/13≈ 0.538553.85%
    ✅ P(M|D) > P(T|D) > P(J|D) → 次品最有可能由 Mary 生产（≈ 53.85%）， 其次 Tom（30.77%），最后 Jane（15.38%）。

    ✔️ 验证：2/13 + 4/13 + 7/13 = 13/13 = 1

    💡 关键洞察：因三人产量相同（先验相等），后验概率完全由次品率决定， 比例恰为 2 : 4 : 7。

    ❓ 问题：一个次品弄坏了机器，它来自供应商 1 的概率是多少？

    先求次品总概率（全概率）：

    P(B) = P(A₁∩B) + P(A₂∩B) = 0.0130 + 0.0175 = 0.0305

    应用贝叶斯定理：

    P(A₁|B) = P(A₁)P(B|A₁) / [P(A₁)P(B|A₁) + P(A₂)P(B|A₂)]
    = (0.65×0.02) / [(0.65×0.02) + (0.35×0.05)]
    = 0.0130 / 0.0305 ≈ 0.4262
    P(A₂|B) = 0.0175 / 0.0305 ≈ 0.5738
    ✅ 虽然供应商 1 供货更多（65%），但因其次品率低，这个次品更可能来自 供应商 2（≈ 57.38%）。

    📝 练习题 41：咨询公司投标

    一家咨询公司为一个大型咨询合同投标，管理层认为中标机会是 50-50。 随后招标方要求提供补充资料。经验表明：在成功中标的投标中有 75% 被要求补充资料， 在失败的投标中有 40% 被要求补充资料。

    • (a) 中标的先验概率（在被要求补充资料之前）？
    • (b) 已知最终中标，被要求补充资料的条件概率？
    • (c) 已知被要求补充资料，求中标的后验概率？
    📌 参考解答

    设 S = 中标成功，U = 失败，R = 被要求补充资料

    (a) 先验概率 P(S) = 0.50（"50-50"）

    (b) 条件概率 P(R|S) = 0.75（题目给定）

    (c) 用贝叶斯定理（P(S)=P(U)=0.5, P(R|S)=0.75, P(R|U)=0.40）：

    P(S|R) = P(R|S)P(S) / [P(R|S)P(S) + P(R|U)P(U)]
    = (0.75×0.5) / [(0.75×0.5) + (0.40×0.5)]
    = 0.375 / (0.375 + 0.20)
    = 0.375 / 0.575 ≈ 0.6522
    ✅ 被要求补充资料后，中标概率从先验的 50% 上升到约 65.22% —— 因为"要补充资料"在成功投标中更常见，这是一个有利信号。

    正态分布（The Normal Distribution）

    📐 概率密度函数（PDF）

    f(x) = 1 / (σ√(2π)) · e^[ −(x−μ)² / (2σ²) ]

    记作 X ~ N(μ, σ²)，μ = 均值，σ = 标准差。

    🔔 关键特征

    特征说明
    形状钟形曲线，关于 μ 对称
    中心均值 = 中位数 = 众数 = μ
    范围x ∈ (−∞, +∞)，曲线下总面积 = 1
    离散度由 σ 控制，σ 越大曲线越扁平

    📊 经验法则（68–95–99.7 Rule）

    范围包含数据比例
    μ ± 1σ≈ 68%
    μ ± 2σ≈ 95%
    μ ± 3σ≈ 99.7%

    🎯 标准化：Z 分数

    把任意正态分布转化为标准正态分布 Z ~ N(0, 1)：

    Z = (x − μ) / σ

    Z 表示某个值距离均值有几个标准差，再查 Z 表得到左侧面积 P(Z ≤ z)。

    🔔 标准正态分布曲线（68–95–99.7 经验法则）

    深色 = μ±1σ（≈68%）；加中色 = μ±2σ（≈95%）；加浅色 = μ±3σ（≈99.7%）。曲线对称于 μ，两端尾部各仅 2.1%。

    📐 μ 与 σ 对曲线的影响

    ① μ（均值）决定位置：曲线左右平移，形状不变。 ② σ（标准差）决定胖瘦：σ 越大越矮胖、越分散；σ 越小越高瘦、越集中。两者共同决定 N(μ, σ²)。

    正态分布 ↔ 标准正态分布的换算

    🔄 两个方向的换算公式

    方向公式用途
    正态 → 标准正态（标准化）Z = (X − μ) / σ已知原始分，求 Z 去查表
    标准正态 → 正态（反标准化）X = μ + Zσ已知 Z（反查表得到），还原成原始分

    两公式是同一等式的移项：Z = (X−μ)/σ 两边乘 σ 再加 μ，即得 X = μ + Zσ。

    📊 X 轴与 Z 轴一一对应（μ=60, σ=12）

    同一条曲线的两套刻度：原始分 60 ↔ Z 的 0，72 ↔ 1，84 ↔ 2……每个原始分都对应唯一的 Z。

    📖 换算的本质（两步坐标变换）

    Z = (1/σ) × (X − μ)   ←  ① 平移（减 μ，中心移到 0）  ② 缩放（除 σ，单位变成"几个标准差"）

    ✏️ 用分班测试数据演示

    正向（求某分数的 Z）：X = 84 ⇒ Z = (84−60)/12 = +2（高于均值 2 个标准差）

    反向（已知 Z 求分数）：Z = 0.52 ⇒ X = 60 + 0.52×12 = 66.24

    🧭 解题时怎么选方向

    题目类型已知 → 求用哪个公式
    "分数高于/低于…的比例"分数 → 概率先 Z=(X−μ)/σ，再查表
    "前/后 X% 的分数线"概率 → 分数先反查表得 Z，再 X=μ+Zσ

    例题：Strathmore 数学分班测试（正态分布应用）

    题目：Strathmore 商学院数学分班测试，平均分 60%，标准差 12%， 假设成绩服从正态分布。

    1. 求成绩满足以下条件的考生比例：(1) 高于 50%；(2) 低于 80%；(3) 在 60%–90% 之间
    2. 求及格线，使 70% 的考生不及格
    3. 求及格线，使 65% 的考生及格

    🪜 Step 1: 用数学语言写出已知

    设 X = 考生分数（%），则 X ~ N(μ, σ²) = N(60, 12²)，μ = 60，σ = 12。

    题文字数学表达
    1-1高于 50%P(X > 50)
    1-2低于 80%P(X < 80)
    1-3在 60%–90% 之间P(60 < X < 90)
    270% 不及格的及格线 k求 k 使 P(X < k) = 0.70
    365% 及格的及格线 k求 k 使 P(X > k) = 0.65

    🪜 Step 2: 标准化随机变量

    Z = (X − μ) / σ = (X − 60) / 12
    题边界 XZ = (X−60)/12Z 值
    1-150(50−60)/12−0.83
    1-280(80−60)/121.67
    1-360(60−60)/120
    1-390(90−60)/122.5

    🪜 Step 3: 查 Z 表求解

    1-1) P(X > 50) = P(Z > −0.83)

    = 1 − P(Z ≤ −0.83) = 1 − 0.20327 = 0.7967

    → 约 79.67% 的考生得分高于 50%。

    1-2) P(X < 80) = P(Z ≤ 1.67)

    = 0.9525

    → 约 95.25% 的考生得分低于 80%。

    1-3) P(60 < X < 90) = P(0 < Z < 2.5)

    = P(Z ≤ 2.5) − P(Z ≤ 0) = 0.99379 − 0.5000 = 0.4938

    → 约 49.38% 的考生得分在 60%–90% 之间。

    💡 区分两类题：上面是"已知分数 → 求比例"（正向：标准化后查表）； 下面第 2、3 题是"已知比例 → 求分数"（反向：先反查 Z，再用 X = μ + Zσ 还原）。

    2) 及格线 k，使 70% 不及格

    70% 不及格 → P(X < k) = 0.70。反查表：面积 0.70 对应 Z ≈ 0.52（表值 0.69847）。

    ✅ 及格线定在约 66.24%，则约 70% 的考生不及格。

    k = μ + Zσ = 60 + (−0.39) × 12 = 60 − 4.68 = 55.32%
    ✅ 及格线定在约 55.32%，则约 65% 的考生及格。

    📝 练习 1：人事测试（Personnel Tests）

    某项灵巧度测试（dexterity test）由一家私人测试机构全国统一施测。去年所有测试的分数 近似服从正态分布，均值 75，标准差 7.5。

    • (a) 某雇主要求应聘者灵巧度测试至少得 80 分。求去年测试分数中超过 80 分的百分比。
    • (b) 测试机构报告某位应聘者的分数落在分布的第 98 百分位 （即约 98% 的分数低于他、只有 2% 高于他）。求该应聘者的分数。
    📌 参考解法

    设测试分数 X ~ N(75, 7.5²)。

    (a) 超过 80 分的比例

    Z = (80 − 75) / 7.5 = 0.6667 ≈ 0.67

    标准正态表给出 Φ(0.67) = 0.7486，因此右尾概率为：

    P(X > 80) = P(Z > 0.67) = 1 − 0.7486 = 0.2514
    ✅ 去年约有 25.14% 的测试分数超过 80 分（用精确计算约为 25.25%）。

    (b) 第 98 百分位对应的分数

    第 98 百分位表示 P(X ≤ x) = 0.98。反查标准正态表，得到 Z ≈ 2.05。

    x = μ + Zσ = 75 + 2.05 × 7.5 = 90.375
    ✅ 该应聘者的分数约为 90.4 分（若取整数，约 90 分）。

    四、置信区间估计（Confidence Interval Estimation）

    🎯 从点估计到区间估计

    点估计（point estimate）用一个样本统计量估计总体参数，例如用样本均值 x̄ 估计总体均值 μ；但不同样本会产生不同结果，因此还需要说明估计的不确定性。

    置信区间（confidence interval, CI）给出一个合理的参数范围，其通用结构为：

    置信区间 = 点估计 ± 误差界限（Margin of Error）
    = 点估计 ± 临界值 × 标准误（Standard Error）
    • Confidence level（置信水平）：构造区间的方法在长期重复抽样中的成功率，常用 90%、95%、99%。
    • Standard error（标准误）：统计量在重复抽样中的标准差，反映抽样波动。
    • Margin of error, E（误差界限）：区间中心到任一端点的距离。
    💡 95% 置信区间的正确解释：如果用同一种方法反复抽样并构造区间，约 95% 的区间会包含真实参数。 不能说“这个已经算出的区间有 95% 概率包含固定的 μ”。

    📊 常用置信水平与 Z 临界值

    双侧区间把 α = 1 − confidence level 平分到左右两尾，每尾为 α/2。

    置信水平α每尾 α/2Zα/2
    90%0.100.051.645
    95%0.050.0251.960
    99%0.010.0052.576

    置信水平越高，临界值越大，区间越宽；样本量越大，标准误越小，区间越窄。

    🧮 置信区间公式的推导

    Step 1：样本均值的抽样分布

    设 X₁, X₂, …, Xₙ 是来自总体的独立同分布随机样本，满足 E(Xᵢ) = μ、Var(Xᵢ) = σ²。样本均值为：

    x̄ = (X₁ + X₂ + ··· + Xₙ) / n

    利用期望与方差的性质：

    E(x̄) = [E(X₁) + ··· + E(Xₙ)] / n = nμ/n = μ

    Var(x̄) = [Var(X₁) + ··· + Var(Xₙ)] / n²
    = nσ²/n² = σ²/n

    SD(x̄) = SE(x̄) = σ/√n

    若总体正态，或 n 足够大使中心极限定理适用，则：

    x̄ ~ N(μ, σ²/n)

    Step 2：标准化

    把样本均值标准化为标准正态变量：

    Z = (x̄ − μ) / (σ/√n) ~ N(0,1)

    中间面积为 1 − α 时，左右两尾各为 α/2，因此：

    P[−Zα/2 ≤ (x̄ − μ)/(σ/√n) ≤ Zα/2] = 1 − α

    Step 3：把不等式改写为关于 μ 的区间

    令 SE = σ/√n。将不等式各部分乘以 SE，再移项：

    P[−Zα/2SE ≤ x̄ − μ ≤ Zα/2SE] = 1 − α

    P[x̄ − Zα/2SE ≤ μ ≤ x̄ + Zα/2SE] = 1 − α

    于是得到总体均值在 σ 已知时的置信区间：

    x̄ − Zα/2 × σ/√n ≤ μ ≤ x̄ + Zα/2 × σ/√n
    即：x̄ ± Zα/2 × σ/√n

    σ 未知时：为什么改用 t 分布？

    用样本标准差 s 估计未知的 σ 会增加不确定性。若总体正态，则统计量：

    T = (x̄ − μ)/(s/√n) ~ tn−1

    沿用相同的不等式推导，可得：

    x̄ ± tα/2,n−1 × s/√n

    t 分布尾部较厚，因此小样本时 t 临界值大于 Z 临界值；n 增大后，t 分布逐渐接近标准正态分布。

    总体比例 p 的近似推导

    样本比例 p̂ 的期望为 p，方差为 p(1−p)/n。大样本下：

    [p̂ − p] / √[p(1−p)/n] ≈ N(0,1)

    由于 p 未知，用 p̂ 估计标准误，得到常用的比例置信区间：

    p̂ ± Zα/2 × √[p̂(1−p̂)/n]
    💡 推导中的概率 1 − α 描述的是随机区间构造方法的长期覆盖率。 抽样完成后，μ 是固定参数，区间要么包含它、要么不包含它。

    1️⃣ 总体均值 μ：总体标准差 σ 已知

    总体近似正态，或样本量足够大且随机、独立时，使用 Z 区间：

    x̄ ± Zα/2 × σ/√n

    商业例题：某配送流程的总体标准差已知为 12 分钟。随机抽取 64 个订单， 平均配送时间为 52 分钟。求总体平均配送时间的 95% 置信区间。

    SE = σ/√n = 12/√64 = 1.5
    E = 1.96 × 1.5 = 2.94
    CI = 52 ± 2.94 = [49.06, 54.94]
    ✅ 我们有 95% 的置信度认为总体平均配送时间位于 49.06 至 54.94 分钟之间。

    Interval Estimate of Population Mean: σ Known

    Wines and Spirits Ltd. 在肯尼亚共有 260 家零售门店。公司正在评估一个新门店的潜在选址， 其中一项依据是新址目标商圈内居民的平均年收入。

    随机抽取一个样本：

    • 样本量：n = 36
    • 样本平均年收入：x̄ = $31,100
    • 总体标准差：σ = $4,500
    1. 求总体平均年收入的 95% 和 99% 置信区间，并解释结果。
    2. 假设你是总经理，在估计总体均值时，哪个区间最适合使用？为什么？
    📌 参考解法

    投影片标题指定 σ known，因此使用总体均值的 Z 区间：

    x̄ ± Zα/2 × σ/√n

    Step 1：计算标准误

    SE = σ/√n = 4,500/√36 = 4,500/6 = $750

    (i-a) 95% 置信区间

    Z0.025 = 1.96
    E = 1.96 × 750 = $1,470
    CI = 31,100 ± 1,470 = [$29,630, $32,570]

    解释：采用这种抽样和构造区间的方法反复进行，约 95% 的区间会包含目标商圈居民的真实平均年收入。

    (i-b) 99% 置信区间

    Z0.005 = 2.576
    E = 2.576 × 750 = $1,932
    CI = 31,100 ± 1,932 = [$29,168, $33,032]

    解释：我们有 99% 的置信度认为目标商圈居民的真实平均年收入位于 $29,168 至 $33,032 之间。

    (ii) 管理者应选哪个区间？

    99% 区间的置信水平更高，但更宽；95% 区间更窄、估计更精确，但置信水平稍低。

    ✅ 一般商业选址可采用 95% 置信区间，因为它在可靠性与精度之间较为平衡。 若错误选址的成本极高、管理层更重视降低遗漏真实均值的风险，则可采用 99% 区间，并关注其保守下限。
    💡 260 家门店不是本题的总体规模，因为样本研究对象是目标商圈居民。若课堂 Z 表把 99% 临界值取为 2.58， 会得到近似区间 [$29,165, $33,035]，与使用 2.576 的结果仅有舍入差异。

    2️⃣ 总体均值 μ：总体标准差 σ 未知

    用样本标准差 s 代替 σ，并使用 t 分布；自由度 df = n − 1：

    x̄ ± tα/2, n−1 × s/√n

    t 分布比标准正态分布尾部更厚，用来补偿用 s 估计 σ 带来的额外不确定性；n 越大，t 越接近 Z。

    例题：抽取 16 名员工，平均培训成绩 x̄ = 75，样本标准差 s = 8。 假设总体近似正态，求 μ 的 95% 置信区间。df = 15 时，t0.025,15 = 2.131。

    SE = 8/√16 = 2
    E = 2.131 × 2 = 4.262
    CI = 75 ± 4.262 = [70.74, 79.26]

    📈 Student's t Distribution（Continued）

    为什么需要 t 分布？

    当总体标准差 σ 未知时，只能用样本标准差 s 估计它。不同样本的 s 也会波动， 因而比 σ 已知时多一层不确定性。Student's t 分布通过较厚的尾部反映这种额外不确定性。

    T = (x̄ − μ) / (s/√n)

    自由度（Degrees of Freedom）

    df = n − 1

    计算 s 时先用样本数据估计了 x̄。在 n 个离均差中，一旦前 n−1 个确定，最后一个必须满足 Σ(Xᵢ−x̄)=0，因此只有 n−1 个可以自由变化。

    t 分布的主要特征

    • 关于 0 对称，形状与标准正态分布相似。
    • 尾部比标准正态分布更厚，因此相同置信水平下 t 临界值通常更大。
    • 每个自由度对应一条不同的 t 曲线。
    • df 越大，s 对 σ 的估计越稳定，t 分布越接近标准正态分布。
    • 当 df → ∞ 时，t 分布趋近 Z 分布。

    📊 t 分布 vs 正态分布（图示）

    df=3（橙）明显矮胖、尾厚；df=10（青）已接近正态；df→∞ 时 t 完全变成 Z。n≥30 后两者几乎重合。

    3️⃣ 总体比例 p 的置信区间

    若样本中有 x 个“成功”，样本比例 p̂ = x/n。正态近似条件通常要求 np̂ ≥ 10 且 n(1 − p̂) ≥ 10。

    p̂ ± Zα/2 × √[p̂(1 − p̂)/n]

    市场调查例题：随机调查 200 名顾客，其中 120 人愿意推荐该品牌。 求总体推荐比例的 95% 置信区间。

    p̂ = 120/200 = 0.60
    SE = √[0.60(0.40)/200] = 0.03464
    E = 1.96 × 0.03464 = 0.0679
    CI = 0.60 ± 0.0679 = [0.5321, 0.6679]
    ✅ 总体中愿意推荐该品牌的顾客比例估计为 53.21%–66.79%。

    估计目标样本量公式说明
    总体均值 μn = (Zα/2σ/E)²σ 可来自历史数据或预调查
    总体比例 pn = Z²α/2p*(1−p*)/E²未知 p 时取 p* = 0.5，得到最保守样本量

    例：95% 置信水平下估计顾客比例，要求误差不超过 5%，且事先不知道 p：

    n = 1.96² × 0.5 × 0.5 / 0.05² = 384.16 → 至少 385 人

    五、误差界限（Margin of Error）

    🎯 定义与直观理解

    Margin of Error（误差界限，记作 E 或 ME）是置信区间的“半宽”， 表示样本估计值与总体真实参数之间允许的最大抽样误差范围。

    置信区间 = 点估计 ± E
    E = 临界值 × 标准误（Standard Error）

    例如，顾客平均消费额估计为 KES 8,000，误差界限为 KES 500，则区间为：

    8,000 ± 500 = [7,500, 8,500]
    💡 E 描述的是抽样误差，不能补偿抽样偏差、测量错误、无应答偏差或错误的数据收集方法。

    📐 常用公式

    估计对象条件误差界限 E
    总体均值 μσ 已知Zα/2 × σ/√n
    总体均值 μσ 未知tα/2,n−1 × s/√n
    总体比例 p正态近似适用Zα/2 × √[p̂(1−p̂)/n]

    若从大小为 N 的有限总体中不放回抽样，且抽样比例较大，可把标准误乘以有限总体修正：

    FPC = √[(N − n)/(N − 1)]

    📝 例题 1：总体均值的误差界限

    某产品交付时间的总体标准差已知为 15 小时。随机抽取 n = 100 个订单， 在 95% 置信水平下求误差界限。

    E = 1.96 × 15/√100
    = 1.96 × 1.5 = 2.94 小时
    ✅ 样本均值对总体平均交付时间的估计精度为 ±2.94 小时。

    📝 例题 2：总体比例的误差界限

    调查 400 名顾客，其中 62% 表示满意。在 95% 置信水平下：

    E = 1.96 × √[0.62(0.38)/400]
    = 1.96 × 0.02427 = 0.04757
    ✅ 误差界限约为 ±4.76 个百分点；95% 置信区间约为 57.24%–66.76%。
    “百分点”与“百分比变化”不同：62% ± 4.76 个百分点直接得到 57.24% 和 66.76%。

    🔄 哪些因素会改变误差界限？

    变化对 E 的影响原因
    置信水平提高E 增大临界值 Z 或 t 增大
    样本量 n 增加E 减小标准误按 1/√n 缩小
    数据变异 σ 或 s 增加E 增大样本均值波动更大
    比例更接近 0.5E 增大p(1−p) 在 p = 0.5 时最大
    • 样本量扩大到原来的 4 倍，E 才会约减半。
    • 在其他条件相同时，99% 区间比 95% 区间更宽，95% 又比 90% 更宽。
    • 总体规模通常不会直接改变 E；只有抽样比例较大时，有限总体修正才明显。

    🧭 从目标误差反推样本量

    研究设计常先规定可接受的 E，再反解样本量：

    均值：n = (Zα/2σ/E)²
    比例：n = Z²α/2p*(1−p*)/E²

    因此 E 越小，所需样本量增长得非常快；所有计算结果必须向上取整。

    ⚠️ 常见错误：把 E 当成标准误；忘记 E 是区间总宽度的一半；比例题把 5% 写成 5 而不是 0.05； 把“±4%”误读为整个区间宽度 4%（实际总宽度为 8%）。

    六、样本量确定（Sample Size Determination）

    🎯 为什么要事先确定样本量？

    样本太小，估计不精确、结论不稳定；样本过大，则浪费时间和成本。 样本量确定的目标是在统计精度与调查成本之间取得平衡。

    用于置信区间估计时，样本量主要由以下因素决定：

    因素变化对所需样本量的影响
    置信水平提高（如 95% → 99%）临界值变大，n 增加
    允许误差 E减小n 按 1/E² 快速增加
    总体变异程度σ 或 p(1−p) 增大n 增加
    总体规模 N总体较小可用有限总体修正降低 n
    预计应答率降低需要联系更多对象
    💡 误差减半通常意味着样本量约增至 4 倍，因为公式中 E 被平方。

    1️⃣ 估计总体均值 μ 所需的样本量

    n₀ = (Zα/2σ / E)²
    • σ：总体标准差，可用历史研究或预调查（pilot study）的 s 估计。
    • E：可接受的最大误差，单位与原变量相同。
    • 计算结果无论小数多小，都必须向上取整。

    例题：估计顾客平均每月消费额，历史标准差约为 KES 18,000。 要求 95% 置信水平，误差不超过 KES 3,000。求最小样本量。

    n₀ = (1.96 × 18,000 / 3,000)²
    = 138.2976 → n = 139
    ✅ 至少需要随机抽取 139 名顾客。
    若 σ 未知，可先做小规模预调查，用样本标准差 s 代替；严格的小样本设计可迭代使用 t 临界值。

    2️⃣ 估计总体比例 p 所需的样本量

    n₀ = Z²α/2 × p*(1 − p*) / E²

    p* 是规划阶段对总体比例的预估：

    • 有历史数据或预调查时，使用相应的 p*。
    • 完全未知时，使用 p* = 0.5；此时 p*(1−p*) 最大，得到最保守的样本量。

    例题 A（有先验估计）：预计约 30% 的顾客会购买新产品。 在 95% 置信水平下，希望误差不超过 4%。

    n₀ = 1.96² × 0.30 × 0.70 / 0.04²
    = 504.21 → n = 505

    例题 B（p 未知）：若仍要求 95% 置信水平和 4% 误差，但没有先验估计：

    n₀ = 1.96² × 0.50 × 0.50 / 0.04²
    = 600.25 → n = 601
    ✅ p 未知时需要更大的保守样本量：601 人。

    3️⃣ 有限总体修正（Finite Population Correction）

    前面的 n₀ 假定总体很大。若从规模为 N 的有限总体中不放回抽样， 且初始样本量超过总体的约 5%，可修正为：

    n = n₀ / [1 + (n₀ − 1)/N]  =  Nn₀ / (N + n₀ − 1)

    例题：公司共有 N = 1,200 名员工。按大总体公式算得 n₀ = 385：

    n = 385 / [1 + (385 − 1)/1,200]
    = 291.67 → 292 人
    ✅ 使用有限总体修正后，最小有效样本量为 292 人。

    4️⃣ 无应答与复杂抽样调整

    无应答调整（Non-response Adjustment）

    若预计应答率为 r，需要联系的人数为：

    ncontact = nrequired / r

    若需要 292 份有效答卷，预计应答率为 80%：

    ncontact = 292 / 0.80 = 365 人

    设计效应（Design Effect, DEFF）

    整群抽样、分层抽样等复杂设计的精度可能不同于简单随机抽样。已知设计效应时：

    ndesign = nSRS × DEFF

    若简单随机抽样要求 385 人，而整群设计的 DEFF = 1.5，则需 385 × 1.5 = 577.5，向上取整为 578 人；之后再按预计应答率调整。

    🧭 完整计算顺序

    1. 明确估计目标：总体均值 μ 还是总体比例 p。
    2. 选择置信水平，并确定 Zα/2。
    3. 设定可接受误差 E。
    4. 取得 σ 或 p* 的规划值；未知比例时用 0.5。
    5. 计算大总体初始样本量 n₀，并向上取整。
    6. 总体较小时应用有限总体修正。
    7. 如采用复杂抽样，乘以设计效应 DEFF。
    8. 最后按预计应答率放大需要联系的人数，并再次向上取整。
    ⚠️ 本章公式针对置信区间估计。若研究目标是检验差异或比较两组，样本量还取决于 显著性水平、统计功效（power）和最小可检测效应，需使用功效分析。

    📝 Question 1：购车问题

    John 的家庭决定购买一辆新车，候选车型为 Ford、Chevrolet 或 Toyota。

    • P(买 Ford) = 0.3
    • P(买 Chevrolet) = 0.4

    (i) 求他们购买 Ford 或 Chevrolet 的概率

    (ii) 求他们购买 Toyota 的概率

    (iii) 已知：P(Ford) = 0.3，P(station wagon) = 0.2，P(Ford station wagon) = 0.5 （⚠️ 注：此处 0.5 似为数据异常，可能应为 0.05），求 P(买 Ford 或 station wagon 或两者)

    📌 参考解法

    (i) Ford 和 Chevrolet 是互斥事件（一次只能买一辆）：

    P(F ∪ C) = P(F) + P(C) = 0.3 + 0.4 = 0.7

    (ii) 三个事件互斥且覆盖所有可能：

    P(T) = 1 − P(F) − P(C) = 1 − 0.3 − 0.4 = 0.3

    (iii) 用加法法则。⚠️ 注意 P(Ford station wagon) = 0.5 不可能 （联合概率不能大于 P(Ford)=0.3），几乎可以确定是 0.05 的笔误。按 0.05 计算：

    P(F ∪ W) = P(F) + P(W) − P(F ∩ W)
    = 0.3 + 0.2 − 0.05 = 0.45

    （若硬按题面 0.5 代入会得到 0.3+0.2−0.5 = 0，显然不合理，故判定为笔误。）

    📝 Question 2：Patricia 面试评估

    某医院考虑 Patricia 担任财务管理关键职位。面试和审核后， 人事经理认为 Patricia 表现良好的概率为 75%，表现不佳的概率为 25%。 根据其他公司的经验，咨询公司在评估财务经理时的条件概率如下：

    实际表现 咨询公司预测「会表现好」 咨询公司预测「不会表现好」
    Manager does well0.90.1
    Manager does NOT do well0.20.8

    (i) 若咨询公司预测 Patricia 表现良好，求该预测正确的概率

    (ii) 若咨询公司预测 Patricia 表现不佳，求该预测正确的概率

    📌 参考解法（贝叶斯定理）

    设 W = Patricia 实际表现好；P = 咨询公司预测 W

    P(W) = 0.75, P(W') = 0.25, P(P|W) = 0.9, P(P|W') = 0.2

    (i) P(W|P) = P(P|W)·P(W) / [P(P|W)·P(W) + P(P|W')·P(W')]
    = (0.9 × 0.75) / (0.9 × 0.75 + 0.2 × 0.25)
    = 0.675 / 0.725 ≈ 0.931

    (ii) P(W'|P') = P(P'|W')·P(W') / [P(P'|W')·P(W') + P(P'|W)·P(W)]
    = (0.8 × 0.25) / (0.8 × 0.25 + 0.1 × 0.75)
    = 0.20 / 0.275 ≈ 0.727

    📝 Case：蓝色出租车肇事案（Bayes' Analysis）

    本案例改编自 S. Salop, "Evaluating Uncertain Evidence" with Sir Thomas Bayes, Journal of Economic Perspectives, Summer 1987.

    1994 年 3 月 1 日，一名男子在过马路时被一辆超速出租车撞倒。 一名目击证人作证说她认为肇事出租车（未停车）是蓝色的。 男子起诉 Blue Cab Company 索赔医疗费用。

    该城市只有两家出租车公司：Blue Cab 和 Green Cab。

    • Green Cab 占该市出租车的 85%，Blue Cab 占 15%
    • 目击证人在类似条件下正确识别出租车颜色的可靠率为 80%

    原告律师据此认定：男子极有可能被蓝色出租车撞到。

    Required：使用贝叶斯分析，律师的结论是否正确？

    📌 参考解法

    设 B = 蓝车肇事，G = 绿车肇事，W = 证人说"蓝"

    P(B) = 0.15, P(G) = 0.85, P(W|B) = 0.8, P(W|G) = 0.2

    P(B|W) = P(W|B)·P(B) / [P(W|B)·P(B) + P(W|G)·P(G)]
    = (0.8 × 0.15) / (0.8 × 0.15 + 0.2 × 0.85)
    = 0.12 / (0.12 + 0.17)
    = 0.12 / 0.29 ≈ 0.414
    🎯 结论：实际是蓝车肇事的后验概率只有约 41.4%， 仍小于 50%！律师的结论不正确。这就是著名的"基础率谬误"—— 虽然证人 80% 可靠，但因为 Blue Cab 本身就少，整体上更可能是绿车被误认。

    📝 Exercise：疾病诊断测试

    某疾病的诊断测试有以下特性：

    • 对实际患病者：正确检测出疾病的概率为 90%
    • 对未患病者：测试报告"未患病"的概率为 0.9
    • 人群中仅 1% 患有该疾病

    从人群中随机选一人接受测试，若测试结果显示她患病， 问：她实际患病的条件概率是多少？

    ✅ 解法（贝叶斯定理）

    定义事件：

    • D = 患有该疾病
    • H = 测试结果显示患病

    已知：

    • P(H|D) = 0.9
    • P(D) = 0.01，P(D') = 0.99
    • P(H|D') = 1 − 0.9 = 0.1（假阳性率）

    应用贝叶斯公式：

    P(D|H) = [P(H|D)·P(D)] / [P(H|D)·P(D) + P(H|D')·P(D')]
    = (0.9 × 0.01) / (0.9 × 0.01 + 0.1 × 0.99)
    = 0.009 / (0.009 + 0.099)
    = 0.009 / 0.108
    = 1/12 ≈ 0.0833
    🎯 即使测试结果显示患病，实际患病概率也只有约 8.33%。 这同样是基础率谬误的经典例子——疾病罕见时，假阳性主导结果。

    📊 A. 集中趋势：Mean / Median / Mode

    度量定义公式特点
    均值 Mean所有值之和 ÷ 个数x̄ = Σx / n（分组：Σfx / Σf）用上全部数据，但易被极端值拉偏
    中位数 Median排序后正中间的值奇数：第 (n+1)/2 个；偶数：中间两数平均不受极端值影响
    众数 Mode出现次数最多的值频数最高者（分组用众数组）唯一适用于分类数据

    例：数据 2, 3, 3, 4, 8 → Mean = 20/5 = 4，Median = 3，Mode = 3（极端值 8 把均值拉高了）。

    🎯 三者关系与偏态

    分布形状三者关系
    对称（正态）Mean = Median = Mode
    右偏（正偏，长尾在右）Mode < Median < Mean
    左偏（负偏，长尾在左）Mean < Median < Mode
    💡 口诀：均值总被"尾巴"拉过去。数据有偏或有离群值时，中位数是更可靠的"典型值"。

    📊 B. 离散程度：方差 Variance 与标准差

    定义：方差衡量数据偏离均值的"平均平方距离"。越大越分散。

    类型公式除以
    总体方差 σ²Σ(x − μ)² / NN
    样本方差 s²Σ(x − x̄)² / (n − 1)n − 1（贝塞尔校正，无偏估计）

    简化式：σ² = Σx²/N − x̄²；标准差 σ = √方差（与原数据同单位，更直观）。

    例：2, 3, 3, 4, 8（均值 4） → Σ(x−x̄)² = 4+1+1+0+16 = 22 → 总体方差 = 22/5 = 4.4，样本方差 = 22/4 = 5.5。

    💡 为什么要平方？直接加偏差 Σ(x−x̄) 永远 = 0（正负抵消）；平方后都变正、不抵消，且放大大偏差。

    📐 D. 置信区间估计（Confidence Interval）— 基本概念

    点估计 vs 区间估计：

    • 点估计：用单一统计量（如 x̄）估计总体参数（如 μ）
    • 区间估计：给出一个区间 + "有多大把握"包含真实参数

    置信区间一般形式：

    点估计 ± 误差边际 = x̄ ± Zα/2 · (σ/√n)
    术语含义
    置信水平 (1−α)如 90%、95%、99%
    误差边际 Margin of ErrorZα/2 × 标准误
    标准误 Standard Errorσ/√n
    📎 这里只是速览；完整的置信区间公式、误差界限、样本量确定与例题见上面的「四、置信区间估计」「五、误差界限」「六、样本量确定」章节。

    📊 E. 更多度量：变异系数、偏度、四分位数

    变异系数 CV（Coefficient of Variation）

    衡量"相对离散度"——标准差占均值的百分比，无单位，可跨不同数据集比较。

    CV = (σ / μ) × 100%

    CV 小 → 数据集中、稳定；CV 大 → 波动大。例：超市销售额 σ=0.9934, μ=7.69 → CV ≈ 12.9%（中等偏低）。

    偏度 Skewness（衡量不对称）

    Pearson 第二偏度 = 3 × (均值 − 中位数) ÷ 标准差
    SK 符号形状三者关系
    SK > 0右偏（正偏）Mode < Median < Mean
    SK = 0对称Mean = Median = Mode
    SK < 0左偏（负偏）Mean < Median < Mode

    四分位数与 IQR（Interquartile Range）

    • Q1 = 第 25 百分位，Q2 = 中位数，Q3 = 第 75 百分位
    • IQR = Q3 − Q1 = 中间 50% 数据的跨度（抗极端值）
    • 离群值规则（1.5 × IQR）：小于 Q1 − 1.5×IQR 或大于 Q3 + 1.5×IQR 的点 = 离群值

    设 D=用药, P(D)=0.10, P(T|D)=0.90, P(T|D')=0.10

    📝 C10. 例题：变异系数（两种坏账催收方法）

    方法CountMeanStDev
    Approach A10052.309.23
    Approach B17069.4711.78
    📌 解答

    (i) CV = (StDev ÷ Mean) × 100%

    CV(A) = 9.23 ÷ 52.30 × 100% = 17.65%

    CV(B) = 11.78 ÷ 69.47 × 100% = 16.96%

    (ii) CV 越小越稳定。CV(B) 16.96% < CV(A) 17.65%，且 B 平均催回额更高（69.47 vs 52.30） → Approach B 更高效（收得多 + 更稳定）。

    注意：直接比 StDev 会误导（B 的 11.78 > A 的 9.23）；因均值不同，须用 CV 做相对比较。

    八、核心知识点

    期望值 Expected Value

    • 定义：随机变量在长期试验中的平均结果，E(X) = Σ xᵢ·P(xᵢ)
    • 线性性质：E(aX) = a·E(X)，E(aX + b) = a·E(X) + b
    • 概率验证：所有概率之和必须等于 1

    方差与标准差 Variance & Std. Deviation

    • 方差 Var(X)：衡量离散程度，Var(X) = E(X²) − [E(X)]²
    • 标准差 σ：σ = √Var(X)，与原数据同单位，实际分析中常用
    • 变异系数 CV：CV = σ/μ，用于比较不同尺度数据的相对波动
    • 线性性质：Var(aX) = a²·Var(X)，Var(aX + b) = a²·Var(X)
